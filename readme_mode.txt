## üìù Headings in README.md

Use `#` symbols to create headings:

```markdown
# H1 - Main Title
## H2 - Section Title
### H3 - Subsection
#### H4
##### H5
###### H6 - Smallest Heading
```

**Example:**

```markdown
# My Project

## Features

### Installation

#### Usage
```
